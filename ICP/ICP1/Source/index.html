GitHub?

GitHub is a web-based hosting service for version control using git. It is mostly used for computer code. It offers all of the distributed version control and source code management functionality of Git as well as adding its own features.

Steps involved

1. Creating Repository

For creation of new repository

  1. click upper right corner ,select New repository.
  2. Rename repository WEB_5590.
  3. Select Initialize this repository with a README.

2. Branch creation

New branch creation

  1. As shown in the below figure, Click the drop down at the top of the file list that says branch: master.
  2. Type a branch name, Avinash, into the new branch text box.
  3. Select the blue Create branch box or hit “Enter” on your keyboard.

3. Commit changes

After committing changes

  • Click the README.md file.
  • select pencil icon in the upper right corner of the file view to edit.
  • In editor , write something
  • Click Commit changes button.

4. Open pull request
